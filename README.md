# Sign Language Detection

Real-time translation system that converts hand sign gestures into text and speech for accessibility, supporting ASL alphabet and common phrases.

## Features

- Real-time hand landmark detection
- ASL alphabet recognition (A-Z)
- Common phrase detection
- Text-to-speech output for translated signs
- Webcam and video file input support

## Tech Stack

Python, MediaPipe, TensorFlow, OpenCV, pyttsx3

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/Sign-Language-Detection.git
cd Sign-Language-Detection
pip install -r requirements.txt
```

### Usage

```bash
python detect.py
```
